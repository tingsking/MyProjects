jQuery-jcImgScroll插件

* 插件说明 及 相关参数

jQuery - jcImgScroll v 1.4
speed:400, //图片切换速度设置，提供easing.js值 或 时间值(mm)
width : 618, //设置图片宽度
height:342, //设置图片高度
offsetX:100, //设置图片间距
setZoom:.5, //微调图片缩小比例
loadClass:"loading", //定义loading CSS CLASS样式
position:"center", //定义小图位置，top，center，bottom
count:5, //设置显示图片的数量
arrow : { width : 110, height : 342, x : 20, y : 0 }, //设置箭头高宽和微调xy偏移
NumBtn : false, //是否开起数字按钮
setNumBtn : { width : 19, height : 18, x : 20, y : 0 }, //设置数字按钮高宽和微调xy偏移
title:true, //是否开起信息功能
setTitle : { border : 0, height : 35, bgColor:"#6ba600", color:"#fff", padding:20, opacity:.8 }, //设置信息功能相关配置
